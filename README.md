# StudentGrade-Tracker
Calculate Student Grade
🎓 Student Grade Management System (Java)

A console-based Java application designed to efficiently manage student academic records. This system allows users to add, update, view, and delete student data while generating a well-structured report card with detailed performance analysis.

🚀 Features
➕ Add new students with multiple subjects
📊 Input and validate marks (0–100 range)
📄 Generate formatted student report cards
🔍 Search student records by roll number
✏️ Update marks easily
❌ Delete student records
📋 View all students with detailed statistics
🧮 Automatic calculation of:
Total Marks
Average Marks
Highest & Lowest Marks
Grade (A+, A, B+, B, C, SNC)
🛠️ Technologies Used
Java
OOP Concepts (Encapsulation, Classes & Objects)
ArrayList for dynamic data storage
Scanner Class for user input
📌 How It Works
User enters the number of subjects and their names
A menu-driven system is displayed
User can perform operations like:
Add student(s)
View all records
Search specific student
Update marks
Delete student
The system generates a structured report card output
